# Quiz App (MERN Project)

A full-stack Quiz Application built using the **MERN stack** (MongoDB, Express.js, React.js, Node.js) that allows users to take quizzes, track scores, and view results in a clean and interactive interface.

## Features
- User authentication (Sign Up / Login)  
- Multiple-choice quizzes with real-time answer validation  
- Score tracking and result summary at the end of each quiz  
- Responsive and modern UI built with React.js  
- RESTful API backend using Node.js and Express.js  
- Data storage using MongoDB  

## Tech Stack
**Frontend:** React.js, HTML5, CSS3, JavaScript  
**Backend:** Node.js, Express.js  
**Database:** MongoDB  
**Version Control:** Git & GitHub
